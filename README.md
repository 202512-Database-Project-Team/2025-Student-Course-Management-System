### 流程
### ==========================================
### 1. 学生端：提交免修申请
### ==========================================
POST http://localhost:10086/exemption/add
Content-Type: application/json

{
  "sid": 14,
  "cid": 10,
  "term": "22-春季学期",
  "reason": "学习只会影响我拔剑的速度！"
}

### ==========================================
### 2. 教师端：查看待审核列表
### ==========================================
GET http://localhost:10086/exemption/findPending

### ⚠️注意：执行完第2步后，去下面结果框里找一下刚才那条记录的 "id" 是多少
### 假设查出来的 id 是 10，请修改下面第3步的 "id": 10

### ==========================================
### 3. 教师端：审核通过
### ==========================================
POST http://localhost:10086/exemption/audit
Content-Type: application/json

{
  "id": 14,
  "status": 1,
  "auditReason": "小伙子可以，做了我想做的，去吧！",
  "auditBy": 6
}

### ==========================================
### 4. 学生端：查看结果 (验证状态是否变了)
### ==========================================
GET http://localhost:10086/exemption/findBySid/14
